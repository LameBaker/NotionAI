# Contributing

## Workflow
1. Read `AGENTS.md` and `.codex/BOOTSTRAP.md`.
2. Update `tasks/current_iteration.md` before coding.
3. Keep commits scoped to a single task.
4. Update docs and state files in the same change set.

## Definition of Done
- Behavior covered by tests.
- ACL safety invariants preserved.
- `PROJECT_STATE.md` and task files updated.
