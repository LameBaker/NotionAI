# Session Checklist

## Before Work
- [ ] Scope is clear.
- [ ] Current task exists in `tasks/current_iteration.md`.
- [ ] Dependencies and environment assumptions are documented.

## During Work
- [ ] Decisions are recorded in `DECISIONS.md`.
- [ ] Architecture changes reflected in `docs/architecture.md`.
- [ ] Access-control changes reflected in `configs/access_policies.yaml` or templates.

## Before Handoff
- [ ] `PROJECT_STATE.md` updated.
- [ ] `tasks/current_iteration.md` status updated.
- [ ] Next steps are explicit.
